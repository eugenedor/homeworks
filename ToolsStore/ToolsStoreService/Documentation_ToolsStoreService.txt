ToolsStoreService - Windows-служба по загрузке XML в БД

Основная точка входа для приложения - метод Main класс Program.
ToolsStoreService основной класс службы.
OnStart - запуск по таймеру загрузки xml при старте
TimerElapsed - происходит по истечении интервала времени, таймер останавливается, 
запускается загрузка FileManager.LoadFiles(), затем таймер снова стартует.
OnStop  - остановка запуска.
