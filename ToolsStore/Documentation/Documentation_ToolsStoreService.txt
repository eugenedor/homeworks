ToolsStoreService - Windows-служба по загрузке XML в БД

Основная точка входа для приложения - метод Main класс Program.

ToolsStoreService.cs - основной класс службы.
OnStart - запуск по таймеру загрузки xml при старте.
TimerElapsed - происходит по истечении интервала времени, таймер останавливается, 
запускается загрузка FileManager.LoadFiles(), затем таймер снова стартует.
OnStop  - остановка запуска.

ToolsStoreService.file (namespace)
FileManager.cs
Загрузка xml-файлов происходит в методе LoadFiles, это головной метод.
В нем в первую очередь происходит получение списка файлов, но это не просто список FileInfo, он обернут в класс в FileWithParam.
FileWithParam кастомизированный класс с параметрами, необходимыми для загрузки.
За получение этого списка отвечает метод GetFiles. Рассмотрим его поподробнее.
